# Settings 功能汇总

## Appearance
- Theme：选择系统 / 深色 / 浅色
- Language：切换界面语言（EN / 中文）

## Skill Paths
- Claude Skills Path：自定义 Claude 技能根目录
- Codex Skills Path：自定义 Codex 技能根目录
- Gemini Skills Path：自定义 Gemini 技能根目录

## Marketplace
- skills_repo.json：显示只读路径，并支持打开
- Marketplace Skills：显示当前缓存的技能数量

## Preferences
- Updates：手动检查更新；可开启启动时自动检查；发现新版本后可一键下载并自动安装

## Version
- 底部显示当前应用版本
