# Conduct 日志 PRD

**版本:** 1.1
**日期:** 2026-01-28

## 1. runtime_log.md
- 运行期日志写入 `runtime_log.md`，用于排查安装/复刻等运行问题。
- 文件存放在系统标准日志目录中，便于定位与运维排查。
- 应用启动时自动创建日志文件。

## 2. 正式发布后的日志位置建议
- **原则**: 采用系统标准日志目录，方便用户定位与运维排查。
- **macOS**: `~/Library/Logs/Conduct/`
- **Windows**: `%APPDATA%\\Conduct\\logs\\`

## 3. 访问方式
- GUI/CLI 提示日志目录路径，或在错误弹窗中给出“查看日志”入口。
